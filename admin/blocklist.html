Quick Templates — Block by Lookup

Don't know the IP of someone you want to block? Use these resources:

  • bgp.he.net — look up a company name → get their ASN(s) and IP ranges
  • ipinfo.io — paste an IP from your access logs → see which company owns it
  • whois.domaintools.com — look up a website → get the hosting IP block

Tip: Watch the Activity Log for suspicious access patterns (e.g. one IP fetching 100+ pages in a minute). Those are the IPs worth blocking.
